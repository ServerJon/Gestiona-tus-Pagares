# GestionaTusPagares

This project branch was generated with [NPX CLI](https://www.electronforge.io/templates/typescript-template).


## Development server

Run `npm start` for a dev server. This command execute this other, `tsc && electron-forge start`.


## Build

Run `ng build` to build the project. The build artifacts will be stored in the `dist/` directory. Use the `--prod` flag for a production build.


## Running unit tests

Run `ng test` to execute the unit tests via [Karma](https://karma-runner.github.io).


## Running end-to-end tests

Run `ng e2e` to execute the end-to-end tests via [Protractor](http://www.protractortest.org/).


## Create Electron Desktop App iOS

electron-packager ./ gestiona-tus-pagares-app --platform=darwin --icon src/favicon.icns

## Create Electron Desktop App Win

electron-packager ./ gestiona-tus-pagares-app --platform=win32 --icon src/favicon.ico
